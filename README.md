# CapuanoLunchMenu

School lunch menu calendar for Capuano. Fetches data from the LinqConnect API and displays it as a monthly HTML calendar, with navigation between months and a print/save-as-PDF button.

## Prerequisites

- Node.js (v18+)

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Build for deployment

```bash
npm run build
```

Output goes to `dist/`.
